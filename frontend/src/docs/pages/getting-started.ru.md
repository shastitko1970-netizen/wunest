# С чего начать

WuNest — клиент для ролевой переписки с LLM. Совместим с SillyTavern: карточки, лорбуки, пресеты, темы импортируются напрямую.

## За 5 минут

### 1. Войди через WuApi

На главной странице — «Войти через WuApi». WuApi (api.wusphere.ru) — это наш gateway к моделям. Аккаунт общий с любым другим wusphere-сервисом.

### 2. Заведи персонажа

**Библиотека → Импорт** — брось PNG-карточку или JSON от SillyTavern. Парсер понимает V2 (`chara`) и V3 (`ccv3`).

Или **Библиотека CHUB** — можно искать публичные карточки и импортировать в один клик.

Или **Создать** — форма с полями Имя / Описание / Характер / Сценарий / Первое сообщение.

### 3. Начни чат

Кликни по карточке → «Чат». Модель выбирается в композере; по умолчанию — баланс из WuApi.

### 4. Настрой генерацию

В шапке чата — иконка с ползунками (`mdi-tune-variant`). Там: температура, top-p, max tokens, системный промпт override, Author's Note. Можно сохранить как preset и переиспользовать.
