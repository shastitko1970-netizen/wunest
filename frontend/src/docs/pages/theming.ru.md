# Оформление и CSS

WuNest поддерживает три уровня кастомизации: UI-тумблеры, ST-совместимые переменные и свой CSS.

## Быстрые настройки

**Настройки → Внешний вид**:

- Размер шрифта (0.75× — 1.4×)
- Ширина чата (40-100%)
- Форма аватара (круглые / квадратные)
- Стиль сообщений (пузыри / плоский / документ)
- Акцентный цвет, цвет фона, текста, рамок
- Фоновая картинка + blur
- Тени, анимации, HTML-рендеринг

Это пишет CSS-переменные на `:root` inline — срабатывает мгновенно без перезагрузки.

## Импорт тем SillyTavern

**Внешний вид → Импорт темы ST** — загружает JSON. Из него извлекаются:

- Цвета: `main_text_color`, `italics_text_color`, `quote_text_color`, `border_color`
- Размеры: `font_scale`, `chat_width`, `blur_strength`
- Стили: `avatar_style`, `chat_display`, `noShadows`, `reduced_motion`
- `custom_css` — применяется как свой CSS

Автоматически выставляется scope `chat` — CSS ST-темы не ломает меню. Если тема содержит правила для общих элементов (`body`, `textarea`, `input`), показывается info-уведомление.

## Свой CSS

**Внешний вид → Свой CSS** — textarea для ручного кода. Поддерживает полный CSS синтаксис.

### Scope — «Куда применять»

Под textarea'ой — тумблер с двумя режимами:

- **Только чат** (default) — CSS оборачивается в `@scope (#chat) { ... }` на современных браузерах, или префиксуется `#chat ` вручную на Firefox. Правила применяются только к элементам внутри чата.
- **На всю апу** — CSS применяется как есть, ко всему UI. Правила для `body`/`input`/`textarea` перекрасят все поля во всех меню.

Рекомендуется **Только чат** для ST тем. **На всю апу** — для WuNest-native тем где ты пишешь селекторы под `.nest-*`.

### Переменные темы

Самый чистый способ переделать цвета — переопределить ST-переменные:

```css
:root {
  --SmartThemeBodyColor: #f0f0f0;       /* основной текст */
  --SmartThemeBorderColor: #3c1e50;     /* рамки */
  --SmartThemeQuoteColor: #ef4444;      /* акцент */
  --SmartThemeBlurTintColor: #0b0818;   /* основной фон */
  --SmartThemeChatTintColor: #130d15;   /* панели */
  --SmartThemeBodyFont: 'Andika';       /* шрифт тела */
}
```

WuNest читает эти переменные через fallback chain — они распространяются везде через CSS inheritance.

### Классы WuNest

Для стилизации сообщений:

| WuNest class        | ST alias        | Описание                     |
|---------------------|-----------------|------------------------------|
| `.nest-msg`         | `.mes`          | Строка сообщения             |
| `.nest-msg-body`    | `.mes_block`    | Тело пузыря (фон, рамка)     |
| `.nest-msg-content` | `.mes_text`     | Текст сообщения              |
| `.nest-msg-name`    | `.mes_name`     | Имя отправителя              |

ID-хуки:

| WuNest ID/class   | ST alias          | Описание                          |
|-------------------|-------------------|-----------------------------------|
| `.nest-chat-scroll` | `#chat`         | Контейнер с историей сообщений    |
| `.nest-chat-input`  | `#send_form`    | Композер (форма ввода)            |
| input textarea      | `#send_textarea`| Поле ввода сообщения              |
| Send кнопка         | `#send_but`     | Кнопка отправки                   |
| Топбар              | `#top-bar`, `.topbar` | Верхняя панель              |

Можно писать стили на любом из двух — они одинаково hit'ят элементы.

### Пример

```css
/* Пузырь сообщения с мягким свечением */
.mes {
  background: rgba(30, 15, 45, 0.65);
  border: 1px solid #3c1e50;
  border-radius: 14px;
  box-shadow: 0 2px 10px rgba(120, 60, 200, 0.15);
}

/* Акцент для имени отправителя */
.mes_name { color: #c485ff; letter-spacing: 0.02em; }

/* Цвет текста в композере */
#send_textarea { color: #f1d1ff; }
```

## Если сломалось

Если свой CSS сломал интерфейс — переключи scope на «Только чат», или используй [безопасный режим](/docs/safe-mode) (`?safe` в URL).
