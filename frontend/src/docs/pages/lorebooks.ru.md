# Лорбуки (World Info)

Лорбук — набор записей, которые динамически вкрапливаются в системный промпт, когда упомянутые ключи появляются в диалоге.

## Создание и импорт

**Библиотека → Лорбуки → Новая** или **Импорт** для JSON-файлов SillyTavern. Поддерживаются массивные (`entries: [...]`) и объектные (`entries: { "0": {}, "1": {} }`) shape'ы.

## Записи

Каждая запись имеет:

- **Ключи** — OR-матчинг. Любое попадание активирует запись.
- **Вторичные ключи** — работают только если отметка «Требует вторичный ключ» (selective) включена. Тогда должен совпасть И первичный, И вторичный.
- **Содержимое** — текст, который подставляется в промпт.
- **Позиция** — до или после карточки персонажа. Также поддерживаются `at_depth`, `before_an`, `after_an` (сохраняются, но ещё не enforced в активаторе).

## Параметры активации (M21)

- **Всегда активна** — запись срабатывает без ключей, на каждом турне.
- **Вероятность** (0-100) — случайный гейт. 0 или 100 = всегда, 1-99 = rand.
- **Целые слова** — матчи только по word boundary. `cat` не сработает в `concatenate`.
- **Регистрозависимо** — учёт case при сравнении.
- **Группа** — в пределах одной группы за проход активируется максимум одна запись (lowest InsertionOrder выигрывает). Полезно для вариативных приветствий.
- **Обходить групповое ограничение** — флаг на записи, всегда включается независимо от группы.

## Рекурсия

- **Не триггерить рекурсию** (`exclude_recursion`) — содержимое записи НЕ попадает в окно следующих проходов. Полезно для «описательного» текста который не должен каскадно триггерить другие записи.
- **Не активироваться рекурсией** (`prevent_recursion`) — запись сработает только на первичном скане истории, но не из-за другой записи.

Активатор делает до 3 рекурсивных проходов. Остановится раньше если ни одна запись не добавилась.

## Порядок и глубина

- **Порядок** (InsertionOrder) — меньше = раньше в промпте. При тай-брейке: индекс книги → индекс записи.
- **Глубина** — сколько последних сообщений сканируем на ключи. 0 = глобальный дефолт книги (обычно 4).

## Привязка к персонажу

Из карточки персонажа → иконка «Лорбуки» → multi-select. При генерации из этого персонажа подтягиваются только привязанные книги.
