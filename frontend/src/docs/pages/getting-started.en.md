# Getting started

WuNest is a roleplay chat client for LLMs. SillyTavern-compatible: cards, lorebooks, presets, and themes import directly.

## In 5 minutes

### 1. Sign in via WuApi

On the landing page — "Sign in with WuApi". WuApi (api.wusphere.ru) is our gateway to LLM providers. The account is shared with any other wusphere service.

### 2. Add a character

**Library → Import** — drop a PNG card or JSON export from SillyTavern. The parser understands V2 (`chara`) and V3 (`ccv3`).

Or **CHUB library** — search public cards and import in one click.

Or **Create** — form with Name / Description / Personality / Scenario / First message fields.

### 3. Start chatting

Click a card → "Chat". Pick a model in the composer; the default draws from your WuApi balance.

### 4. Tune generation

In the chat header — the sliders icon (`mdi-tune-variant`). Inside: temperature, top-p, max tokens, system prompt override, Author's Note. Save as a preset and reuse it.
