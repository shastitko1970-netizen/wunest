# Персонажи

## Форматы импорта

WuNest принимает карточки в двух форматах:

- **PNG с tEXt-чанком** — стандартный формат SillyTavern. Метадата в base64 под ключом `ccv3` (V3) или `chara` (V2).
- **JSON** — экспорт из ST или совместимых инструментов. Поддерживаются три shape'а:
  - V3 wrapper: `{"spec":"chara_card_v3","data":{...}}`
  - V2 wrapper: `{"spec":"chara_card_v2","data":{...}}`
  - V2 flat: `{"name":"...","description":"...",...}`

BOM в начале JSON-файла съедается автоматически.

## Импорт

**Библиотека → Импорт** — drop zone, принимает и PNG, и JSON. Бэкенд определяет формат по magic bytes (`89 50 4E 47` для PNG, `{` для JSON).

Встроенный lorebook (`data.character_book.entries`) при импорте автоматически становится отдельным лорбуком и цепляется к персонажу.

## CHUB библиотека

**Библиотека → Библиотека CHUB** — поиск по публичным карточкам с chub.ai. Фильтры: язык, пол, жанр, цель. NSFW тумблер отдельно. Пагинация через стрелки Prev/Next.

Импорт в один клик — фетчим PNG, парсим, сохраняем, возвращаем в твою библиотеку.

## Создание с нуля

**Библиотека → Создать** — форма с тремя секциями:

- **Идентификация:** имя, теги, URL аватара
- **Профиль:** описание, характер
- **Сцена:** сценарий, первое сообщение

Всё сохраняется в `data` как V3 структура, экспортируется обратно в JSON без потерь.

## Редактирование

Карточку персонажа можно открыть и изменить поля. Лорбуки привязываются отдельно — иконкой «Лорбуки» в карточке.
