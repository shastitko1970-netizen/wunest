# Аккаунт и биллинг

Страница `/account` показывает всё что касается твоего профиля в WuNest и WuApi: баланс wu-gold, тариф, активность по моделям, историю золотых транзакций.

## Личные данные

В шапке страницы — имя или username из WuApi (тянется через `/api/me` при каждом заходе). Под ним — дата регистрации в системе. Эти данные не редактируются здесь — управление профилем (имя, аватар, OAuth-привязки) живёт в кабинете WuApi.

Кнопка **«Обновить»** перезагружает все блоки (профиль, статистику, золото) из бэкенда — пригодится после top-up или после ручной правки в WuApi-кабинете.

Кнопка **«Выйти»** — sign-out: чистит cookie `wu_session` (на всех subdomain'ах `*.wusphere.ru`) + best-effort POST в WuApi для серверной инвалидации сессии. После — hard-reload на `/`, AppShell видит «не authenticated» и показывает экран входа.

## KPI-блок (4 карточки)

### Баланс золота
**wu-gold** — внутренняя валюта WuNest/WuApi. 1 gold = $1 USD по курсу провайдера. Хранится с 9-знаковой точностью (nano-units), отображается с 8 знаками после запятой («1.23456789»).

Каждый запрос к моделям через WuApi-pool списывает реальную стоимость в USD от upstream-провайдера (OpenRouter / Anthropic / Google etc.) → конвертируется в gold с округлением half-away-from-zero. **Минимальный платный запрос всегда ≥ 1 nano** — крошечные cost'ы не округляются молча в ноль.

Кнопка **«Пополнить →»** ведёт в WuApi-кабинет (`wusphere.ru/dashboard`), где топ-ап делается через ЮMoney/Telegram Stars/Crypto.

### Тариф
Free / Plus / Prem / Pro Plus / Max. Free даёт минимум — daily-limit запросов в сутки + ограниченный список моделей. Платные тарифы убирают daily-limit и открывают premium модели (Claude Sonnet 4.5, GPT-4o, Gemini Pro, Wu-named тюны).

Под названием тарифа — дата истечения (`до 2026-05-25`), либо «без срока» (lifetime / admin-grant). Кнопка **«Управление →»** ведёт в кабинет WuApi на страницу подписок.

### Сегодня
Сколько запросов сделано за сегодня + сколько осталось из daily-limit'а. Tab отдельно для wu-gold-чатов (через WuApi pool) и BYOK-чатов (твои собственные ключи). BYOK не считается в daily-limit — биллинг полностью на стороне провайдера.

### Рефералы
Сколько людей пришло по твоей ссылке. Реферальные начисления — отдельная фича WuApi, тут только counter.

## Использование токенов
График/таблица за день / неделю / месяц / всего. Считаются `tokens_in + tokens_out` с момента создания аккаунта.

## Чаще всего (Top Models)
Топ-6 моделей по количеству твоих запросов. Для каждой — счётчики «сегодня / за неделю / всего». Полезно понять что ты реально используешь — иногда выясняется что одна модель забирает 80% запросов и нужен BYOK для неё.

## История золота
Таблица последних 50 транзакций. Колонки:
- **Дата** — когда списано/начислено
- **Описание** — kind: `purchase` / `spend` / `refund` / `admin_grant` / `promo` / `dedupe_replay`
- **Модель** — для spend-транзакций видно какая модель потрачена
- **Tokens** — in/out
- **Δ** — изменение баланса (+/-) в gold
- **Баланс после** — состояние после транзакции

История live до **24h ttl** на нашей стороне — старше сгорает с сервера, но **wu-gold-баланс никуда не девается**, ledger полный лежит в WuApi.

## Доступ к WuNest (Beta)

WuNest пока в закрытой бете — пользоваться чатами могут только юзеры с валидным `nest_access_granted` флагом. Если флага нет — на странице видна форма ввода кода активации:

- Код одноразовый (24 char hex), генерируется админом батчами
- На redeem — флаг flip'ается на твоём WuApi-юзере + код помечается использованным
- После redeem поле сворачивается в маленький badge «Доступ активирован»

Если ты не пришёл по реф-ссылке от другого beta-юзера и не получил код — напиши в [@wuapi_support](https://t.me/wuapi_support).

## Восстановление темы (если интерфейс сломался)

Если импортированная ST-тема или твой кастомный CSS сломали интерфейс настолько, что не получается до Settings докликнуть — на Account есть аварийные кнопки:

- **«Сбросить кастомный CSS»** — стирает поле `customCss` в твоём Appearance (фон, цвета, темa остаются)
- **«Безопасный режим»** — перезагружает страницу с `?safe=1` параметром. Скипается ВСЁ: preset CSS, customCss, bg image, persisted Vuetify theme. Возвращает в bundled-default (nestDark) пока ты не уберёшь параметр

Подробнее про safe mode → [`safe-mode`](safe-mode).

## Удалить аккаунт

Удаление аккаунта — на стороне WuApi (`wusphere.ru/dashboard` → Settings → Delete Account). При удалении WuApi-юзера каскадно удаляются все WuNest-данные (chats, characters, BYOK-ключи, presets, lorebooks).

Если хочешь только сбросить WuNest-данные но оставить WuApi-аккаунт — пока такого режима нет, напиши в саппорт.
